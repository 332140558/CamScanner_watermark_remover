# CamScanner_watermark_remover
移除扫描全能王pdf文件水印
# 如何使用
1、在文件夹中找到并双击 run_tool.bat。

如果是第一次运行，它会自动配置环境并安装必要的库（需要几秒钟）。

2、在弹出的窗口中：

点击“浏览”选择你的 PDF 原文件。

确认保存路径（默认在原文件同目录下）。

点击“开始去水印”。

3、处理完成后，打开新生成的 PDF 查看效果。

核心代码逻辑 (main.py)

该工具使用 PyMuPDF 库，原理是在每一页的右下角绘制一个白色矩形来遮盖水印。

# 更新内容 20251201

界面升级：现在工具界面分为两个选项卡（Tab）：

去水印：原有的去水印功能。

PDF拆分：新增的拆分功能。

拆分功能：

选择一个 PDF 文件。

点击“开始拆分”后，程序会在原文件所在目录下创建一个名为 [文件名]_split 的新文件夹。

每一页会被单独保存为 1.pdf, 2.pdf, 3.pdf 等。
